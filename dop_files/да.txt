Описание идеи: Часто в течении дня есть время, когда нечем заняться. 
Проблема решается многими споосбами. 
Один из них - поиграть в игры, на пример города, крестики-нолики или даже лабиринт.
Для этого мы создали бота в теллеграме.

Описание реализации: Зечез библиотеку python-telegram-bot был создан бот под названием "да".
В диалоге с ним можно играть в крестики-нолики, города, лабиринт. 
Так же есть возможность посмотреть рейтинг. 
Реализовано 8 команд. 
Для удобства сохраниения рейтинга есть база данных с orm-моделями.
Используется сторонее API - static-maps.yandex.
В каждой игре есть работа с изображениями.
Присутствует хостинг.
Осуществляется работа с контекстом пользователя.

Процесс работы
Доступны следующие команды:
- /start включить бота
- /menu открыть меню
- /rating посомотреть рейтинг (all, maze, xo, cities - 
- /xo играть в крестки-нолики
- /cities играть в города
- /maze играть в лабиринт
- /end_play закончить досрочно игру
- /stop выключить бота
